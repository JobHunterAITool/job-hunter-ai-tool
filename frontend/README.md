# Job Hunter AI Tool -- Frontend

This repository contains the React + Vite frontend for the **Job Hunter
AI Tool** capstone project.\
The frontend allows users to enter job search information, send requests
to the backend API, and view ranked job matches returned by the system.

The goal of this interface is to provide a simple way for users to
submit job search information and quickly see relevant job results
generated by the backend machine learning service.

## Overview

The frontend collects user input through a search form and sends that
information to the backend API.\
Once the backend processes the request, the frontend displays the
returned job results in a readable format.

The application was created using **React** for the component-based
interface and **Vite** for fast development builds and hot module
replacement.

## Getting Started

## Prerequisites

Before running the frontend, make sure the following are installed:

-   Node.js
-   npm

You can check with:

node -v\
npm -v

## Installation

Install the project dependencies:

npm install

## Running the Development Server

Start the development server with:

npm run dev

After starting the server, Vite will provide a local address, typically:

http://localhost:5173

The development server supports hot module replacement, meaning the page
will update automatically when changes are made to the code.

##Mock API (Development)

During early development, the frontend uses a mock API response to simulate results from the backend. This allows the interface and job results components to be tested before the backend API integration is complete.

The mock data is stored locally and returned to the UI when a search request is submitted.

Example mock data location:

src/mock/SearchResults.json

When the backend service is fully connected, the frontend will send requests to the real /search endpoint instead of using the mock responses.
